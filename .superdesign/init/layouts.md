## Approved layout contract

The product uses a persistent Handleplan shell. Desktop has a restrained top navigation with the Handleplan identity, Planlegg and Oppdag destinations, and compact local-preference controls. Mobile uses the same hierarchy with a compact header and bottom navigation. The basket and active plan persist between Planlegg and Oppdag.

The UI must feel like a trustworthy Norwegian public utility: calm, practical, transparent, and optimized for repeated weekly use rather than marketing conversion.
