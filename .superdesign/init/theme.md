# Theme Inventory

## Repository state

No implementation theme, Tailwind configuration, CSS variables, fonts, or brand assets exist. The visual system will be established in `.superdesign/design-system.md` and mirrored in `.superdesign/context/globals.css` before the first design command.

## Product constraints

- Norwegian locale and plain-language labels.
- WCAG 2.2 AA target with strong focus indication and non-color status cues.
- Mobile-first responsive behavior with comfortable grocery-store tap targets.
- No retailer colors as primary brand colors; retailer identity may appear only as clearly labeled source data.
- The slider must have an equivalent named-list interaction.
- Dense enough for basket comparison, but not spreadsheet-like.
